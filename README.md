# 🧠 AI Face Recognition Attendance System

A smart, automated **attendance management system** using AI-based face recognition technology. This project replaces manual attendance with a fast, accurate, and contactless solution.

---

## 🚀 Features

- 👤 Real-time Face Recognition (LBPH Algorithm)
- 📸 Face Dataset Creation (Image Capture Module)
- 🧾 Automatic Attendance Marking
- 🗃️ SQLite Database Integration
- 📊 Attendance Dashboard Support
- 🕒 Date & Time Logging
- ⚡ Fast & Lightweight Processing
- 💻 Simple GUI (Tkinter / CustomTkinter)

---

## 🛠️ Tech Stack

- Python 🐍
- OpenCV 👁️
- NumPy 🔢
- SQLite 🗄️
- Pandas 📊
- Tkinter / CustomTkinter 🖥️

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/AI-Attendance-System.git
cd AI-Attendance-System
2. Install dependencies
pip install -r requirements.txt
3. Run the project
Step 1: Create Dataset
python dataset_creator.py
Step 2: Train Model
python train_model.py
Step 3: Start Attendance System
python attendance.py
Step 4: View Dashboard
python dashboard.py
